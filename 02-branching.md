## What is Branching?
Branching in Git allows developers to create a separate workspace to work on new features or changes without affecting the main project.

## Why use Branching?
- Keeps the main project safe
- Allows testing of new features
- Makes collaboration easier
- Helps manage different versions of code

## Important Branching Commands

### Create a new branch
git branch branch-name

### View all branches
git branch

### Switch to another branch
git checkout branch-name

### Create and switch to a new branch
git checkout -b branch-name

### Merge a branch
git merge branch-name

### Delete a branch
git branch -d branch-name
